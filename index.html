🎯 발로란트 매치 트래커

디스코드 봇으로 발로란트 커스텀 매치 정보를 추적하세요

디스코드 봇 상태: 온라인

발로란트 커스텀 게임의 매치 정보를 실시간으로 추적하고 디스코드에서 확인할 수 있습니다.

⚠️현재 제한사항

Personal API Key 사용 중

  • ✅ 계정 등록: 정상 작동
  • ❌ 매치 정보 조회: Production API Key 필요
  • 💡 Production Key 신청 시 모든 기능 이용 가능
📝

계정 등록

라이엇 ID를 등록하여 발로란트 계정과 연동할 수 있습니다. 게임명#태그 형식으로 간편하게 등록하세요.

🎮

커스텀 게임 추적

커스텀 게임 세션을 생성하고 관리할 수 있습니다. 방장이 게임을 만들면 자동으로 추적이 시작됩니다.

📊

매치 정보 분석

킬/데스/어시스트, 에이스 횟수, 스파이크 설치 등 상세한 매치 통계를 확인할 수 있습니다. (Production Key 필요)

🏆

MVP 선정

각 게임의 최고 성과자를 자동으로 선정하고 팀별 성과를 한눈에 볼 수 있습니다.

🌍

다중 지역 지원

아시아-태평양, 북미, 유럽, 브라질 등 전 세계 발로란트 서버를 지원합니다.

⚡

실시간 추적

게임 종료 즉시 최신 매치 데이터를 가져와 실시간으로 결과를 확인할 수 있습니다.

📋 사용 가능한 명령어

/등록 라이엇 계정을 등록합니다 (게임명#태그 형식)
/커스텀게임 새로운 커스텀 게임 세션을 생성합니다
/매치정보 최근 매치 정보를 표시합니다 (Production Key 필요)
/상태확인 봇과 API 상태를 확인합니다 (관리자 전용)
/도움말 모든 명령어와 사용법을 확인합니다

🚀 설정 가이드

1
Discord Bot 초대
관리자에게 봇 초대 링크를 요청하여 서버에 봇을 추가하세요.
2
계정 등록
/등록 [게임명#태그] 명령어로 발로란트 계정을 등록하세요.
3
커스텀 게임 생성
/커스텀게임 [게임명]으로 새로운 게임 세션을 만드세요.
4
게임 플레이
발로란트에서 커스텀 게임을 플레이하세요.
5
결과 확인
게임 종료 후 /매치정보로 상세 결과를 확인하세요. (현재 제한됨)

🔓 Production API Key 신청

1
Riot Developer Portal 접속
developer.riotgames.com에서 로그인하세요.
2
Application 생성
"Products" → "Riot Games API"에서 새 애플리케이션을 등록하세요.
3
API 권한 신청
Valorant Match API 접근 권한을 신청하고 사용 목적을 기재하세요.
4
승인 대기
Riot Games의 검토 후 승인되면 Production Key를 발급받을 수 있습니다.

이 프로젝트는 Riot Games API를 사용합니다.
